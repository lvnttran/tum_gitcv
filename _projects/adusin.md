As manufacturing transitions toward **Industry 4.0**, factories require intelligent systems capable of collecting production data, analyzing operational performance, and supporting data-driven decision-making. Ceramic manufacturing presents additional challenges due to complex production processes, equipment variability, and quality control requirements.

**AdUsin** was developed as the implementation platform for my undergraduate graduation thesis at **Da Nang University of Science and Technology**. The project proposes a smart factory framework that integrates Industrial Internet of Things (IIoT) concepts with machine learning to support digital transformation in ceramic manufacturing. The application demonstrates how production data can be collected, visualized, and analyzed to assist monitoring and operational decision-making.

Built using **Python** and the **Streamlit** web framework, AdUsin provides an interactive interface for industrial data exploration and serves as a proof-of-concept for applying intelligent analytics within a smart manufacturing environment.

### Project highlights

- Developed as an undergraduate graduation thesis (Mar 2022 – Mar 2023).
- Proposed a smart factory framework for ceramic manufacturing.
- Integrated Industry 4.0 and IIoT concepts into a practical web application.
- Implemented interactive dashboards using **Streamlit**.
- Demonstrated the use of machine learning for industrial data analysis and decision support.
