Planning trips often means juggling maps, notes, spreadsheets, screenshots, and countless browser tabs. I wanted a lightweight application that keeps everything in one place while remaining **fully offline**, making it useful before and during a trip without relying on cloud services.

**Offline Travel Planner** is a personal side project developed primarily for myself and friends. Rather than following commercial travel apps, the goal is to experiment with ideas, rapidly prototype new features, and create an enjoyable planning experience. The project continues to evolve as I discover new travel habits and brainstorm improvements after every trip.

Built with **Python (Flask)** and **Leaflet.js**, the application stores each destination in its own local database, allowing independent travel plans without requiring user accounts or an internet connection.

### Current features

- Create and manage multiple destinations.
- Independent database for each destination.
- Interactive offline map with draggable itinerary stops.
- Organize trips by day and chronological schedule.
- Route visualization between planned locations.
- Cost estimation and travel summary.
- Rich notes for each destination and stop.
- Custom map labels and annotations.
- Export trip maps as shareable images.
- Responsive web interface for desktop and tablet use.

### Ideas explored

Since this is a personal playground project, I continuously experiment with features that make trip planning more enjoyable rather than more complicated. Some ideas that have been implemented—or are currently being explored—include:

- 🎬 **Story Mode** that automatically plays the itinerary as a cinematic map tour.
- 📊 Trip statistics generated from the itinerary, including total distance, budget, and stop summaries.
- 🚶 Animated route visualization with transportation-aware map markers.
- 🎨 Beautiful trip poster generation for sharing with friends.
- ✨ Smooth map animations, transitions, and interactive visual effects.
- 📍 Flexible annotation system for reminders and planning notes.
- 🔀 Quick brainstorming tools for exploring alternative schedules and activities.
